# Expense Tracker System

## Project Overview

The Expense Tracker System is a Python-based application developed as part of the DecodeLabs Industrial Training Program. The project allows users to record multiple expenses and calculates the total amount spent. It demonstrates the use of fundamental Python programming concepts such as variables, loops, conditional statements, arithmetic operations, and exception handling.

## Objective

The objective of this project is to understand data accumulation and numerical processing by continuously accepting expense entries from users and maintaining a running total.

## Features

* Accepts multiple expense entries from the user.
* Calculates and updates the total expense amount in real time.
* Allows users to stop data entry using a termination keyword.
* Displays the final total spent.
* Handles invalid input using exception handling.

## Technologies Used

* Python 3.x

## Concepts Implemented

* Variables and Data Types
* User Input and Output
* Loops (while loop)
* Conditional Statements
* Accumulator Pattern
* Mathematical Operations
* Exception Handling (try-except)

## How to Run the Project

1. Open a terminal or command prompt.

2. Navigate to the project folder:

   cd "expense-tracker system"

3. Run the Python file:

   python expense_tracker.py

4. Enter expense amounts when prompted.

5. Type `done` to finish and display the total amount spent.
